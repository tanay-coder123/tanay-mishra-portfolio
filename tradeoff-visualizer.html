Engineering Trade-Off Visualizer

Interactive Web-Based Engineering Analysis Tool

Project Overview

The Engineering Trade-Off Visualizer is a web-based application designed to model real-world engineering decision-making. Users can define design parameters, allocate weighted priorities, and instantly visualize the impact of those choices through live charts and evaluations.

Key Features

  • Real-time sliders for speed, torque, and cost
  • User-defined priority weighting (normalized to 100%)
  • Live bar graph visualization
  • Automated scoring and design evaluation
  • Design saving and ranking system

Technologies Used

  • HTML & CSS
  • JavaScript (DOM manipulation, canvas visualization)
  • GitHub Pages for deployment
