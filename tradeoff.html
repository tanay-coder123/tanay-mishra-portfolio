Engineering Trade-Off Simulator

Python • Object-Oriented Programming • Engineering Analysis

Project Overview

The Engineering Trade-Off Simulator is a Python-based tool inspired by FIRST Robotics Competition (FRC) design decisions. It evaluates multiple mechanical design options using weighted performance metrics such as speed, torque, and cost.

Problem Motivation

In robotics and engineering design, improving one variable often comes at the expense of another. This project was created to model those trade-offs programmatically and to practice object-oriented programming while applying real engineering constraints.

How It Works

  • User inputs ratings for speed, torque, and cost (1–10 scale)
  • Each design is represented as a Design object
  • Weighted scoring evaluates overall performance
  • Constraints (minimum torque) affect evaluation outcomes
  • Designs are ranked and analyzed comparatively

Technical Concepts Used

  • Object-Oriented Programming (classes, methods, attributes)
  • User input validation
  • Algorithmic scoring & ranking
  • Engineering trade-off analysis

Future Improvements

  • Customizable weighting profiles for different design goals
  • Graphical output of trade-off comparisons
  • Web-based interface version
